# Guide

This monorepo demonstrates a production-grade setup for publishing Node.js TypeScript packages.

## Packages

- **[@hg-argo/demo-core-lib](/demo-core-lib/)**: Core utilities: random number generation and numeric range helpers.
- **[@hg-argo/demo-game-lib](/demo-game-lib/)**: A guess-the-number game engine built on `demo-core-lib`.
